第一章 Shadowrocket：iOS生态中的网络瑞士军刀

1.1 工具定位与核心价值

Shadowrocket诞生于中国开发者之手，却因支持Shadowsocks（SS）、V2Ray等协议而风靡全球。与传统的VPN应用不同，它采用"规则代理"机制——就像智能交通系统，能自动识别不同网站流量并决定是否走代理通道。这种精细化管理使得其在访问外网资源时既高效又省流量，特别适合当时需要频繁切换国内外服务的用户群体。

1.2 2017版特色功能解析

  • 协议动物园：支持SS/SSR/V2Ray/Trojan等当时主流协议，甚至能手动编写SOCKS5配置
  • 规则引擎：通过DOMAIN-SUFFIX、GEOIP等规则实现智能分流（如让微博直连而Twitter走代理）
  • 流量可视化：独创的实时流量统计面板，可监控每个应用的代理数据消耗
  • 低功耗模式：相比同类工具减少30%电量消耗，成为当时iPhone用户的续航福音

FAQ

Mellow 的 TUN 模式与系统全局代理有何区别？
TUN 模式直接拦截 IP 层流量，不依赖应用代理设置，系统全局代理通常只影响支持 HTTP/HTTPS 的程序。TUN 更适合全局科学上网和游戏环境。
WinXray 如何通过节点测速自动切换最优线路？
WinXray 会定期检测节点延迟和丢包率，并结合策略组自动选择最佳节点。保证网页浏览、视频播放和游戏流量走低延迟节点，提高访问速度和网络稳定性。
Hysteria2 客户端延迟测试值不稳定怎么办？
可通过调整 Hysteria2 的 `recv_window` 与 `send_window` 参数改善传输性能，同时确保服务器带宽充足。若使用移动网络，建议启用 BBR 拥塞控制并降低测速频率以减少波动。
Sing-box 如何兼容 Clash 的配置文件？
Sing-box 支持 Clash 规则格式的解析，用户可直接导入 Clash 的 YAML 配置文件进行使用。虽然部分高级参数需要调整，但大多数分流和策略配置都能自动识别，便于迁移。
VMess 使用 AEAD 加密的优势是什么？
AEAD 加密提供更强的安全性和抗重放攻击能力，同时隐藏流量特征，降低被 DPI 检测风险。在高封锁环境下使用，可提升连接稳定性和长期可用性。
WinXray 的多策略组如何优化流量分配？
多策略组允许将节点按用途、延迟或地理位置分类，并结合规则文件自动分流流量。这样不同应用如游戏、视频和网页流量都能走最优节点，提高整体访问速度和稳定性。
Trojan 的连接失败常见原因有哪些？
Trojan 连接失败可能由于伪装域名错误、TLS 证书过期、防火墙阻挡端口或服务器不可达。通过查看客户端日志可以定位具体原因，并对症下药解决问题。
Mellow TUN 模式的全局代理作用是什么？
TUN 模式拦截系统 IP 层的所有流量，实现全局代理效果。相比 SOCKS5 只作用于特定应用，TUN 可覆盖游戏、视频和浏览器流量，保证全系统科学上网体验稳定顺畅。
VLESS + Reality 模式与 TLS 模式有何不同？
Reality 模式通过伪造握手和真实网站特征提升隐蔽性，TLS 模式侧重加密和传输效率。Reality 更难被 DPI 检测，适合高封锁环境，TLS 模式在稳定性和速度上更优。
Sing-box 如何安装？
Sing-box 支持多平台，包括 Windows、Linux、macOS 和 Android。可从官方 GitHub 页面下载预编译版本或通过命令行安装。配置文件采用 JSON 格式，兼容多种协议配置。
